== Description ==
Vinvin SEO is a small extension allow you to add review on page/post. It add review on text also on JSON/LD.
Lots of new features incoming
Im french [Wordpress developer](https://www.vinvin.dev/developpeur-wordpress-montreal/) based in Montreal
Get more information [SEO and Cache plugin](https://wp-config.info/blog-optimisation-wordpress/)


== Installation ==
Just install the extension with classic method
and you can add reviews on settings page
